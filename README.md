
# RK CSC Center

## 📌 About Project
RK CSC Center is a Python-based project designed to manage Common Service Center (CSC) activities efficiently.  
It helps in handling users, services, and records in a simple and organized way.

---

## 🚀 Features
- User Management
- Service Handling
- Record Management
- Simple and Clean Interface
- Easy to Use

---

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS

---

## 📂 Project Structure
- core/ → Main project settings
- rk_csc_centre/ → Main app
- templates/ → HTML files
- static/ → CSS files

---

## ▶️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/rk-csc-center.git
   ```

2. Go to project folder:
   ```bash
   cd rk-csc-center
   ```

3. Run server:
   ```bash
   python manage.py runserver
   ```

4. Open browser:
   ```
   http://127.0.0.1:8000/
   ```
